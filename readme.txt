Projet de fin d'UE Recherche Opérationelle

Instructions d'utilisation :
Dans le REPL Julia, importer le fichier main.jl, lancer la fonction solveCovid19(nomFichier) avec nomFichier étant le fichier contenant les données du problème.
Pour des tests plus approfondis, on peut importer le fichiers test.jl et utiliser la fonction de test (EN modifiant la variable n dans le code à sa guise).
